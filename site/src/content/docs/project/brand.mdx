The mark is a terminal prompt `>_` set in a blue→sky gradient tile, with a green "live" dot in the corner — the three ideas agent-presence is built on: the **terminal** it lives in, **presence** (who's working), and a **live** signal.

<BrandShowcase download="Download SVG" colors="Colors" type="Type" />

## Using the mark

- Keep clear space around the tile of at least the green dot's diameter.
- Don't recolor the gradient, rotate the tile, or stretch it.
- It carries its own tile, so it reads on both light and dark surfaces — no extra container needed.
