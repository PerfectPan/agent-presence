Price supported models from a committed LiteLLM snapshot instead of relying only on the small hand-written fallback table. The snapshot is generated by `pnpm run update-pricing`, kept intentionally small via a supported-model allowlist, and refreshed by a weekly GitHub Action PR.

This fixes usage cost estimates for models already seen in supported sources, including `gpt-5.5`, `claude-fable-5`, `deepseek-v4-pro`, and `gemini-3-flash-preview`, while preserving `config.usage.pricing` overrides and `n/a` for unknown models.
